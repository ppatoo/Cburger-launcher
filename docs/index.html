Creating a Custom Link

Method 1: With a domain.

To create a custom link using a domain, follow these steps:

  1. Open your domain registrar's DNS management settings
  2. Create a new record with type "CNAME", and the value "dns.webmc.xyz"
    3. TIPS: You can also create a type "A" record with the value "194.164.76.108" and an "AAAA" record with the value "2a02:4780:28:2a9::1"
    If using     cloudflare, You can get a free SSL certificate by making sure that the icon is colored, not
  3. Just wait up to 48 hours and you should be able to visit your domain and see the WebMC Launcher

FAQ

  • Q: Can I use the WebMC Launcher with any Minecraft server version?
    A: You can only join a server corresponding to your Eaglercraft version, but if the server is running ViaRewind then you can join newer servers, such as 1.21.
  • Q: Can I play 1.21?
    A: No. Any version past 1.17 is near impossible to make work in a web browser, but 1.12.2 support is coming soon.
